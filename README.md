# Микросервис для управления домашними заданиями для учеников

В данном микросервисе преподаватель может назначать группе учеников домашние задания. Домашнее задание включает в себя текст задания, урок, к которому данное задание относится, приложенные ссылками материалы, срок сдачи, количество попыток и назначенную группу учеников.  
Преподаватель может изменить текст и срок сдачи, а так же полностью отменить домашнее задание.  
Преподаватель может просматривать выполненные домашние задания учеников.
Преподаватель может менять статус решения домашнего задания и менять оценку.

Так же в данном микросервисе ученик, состоящий в определенной группе, может просмотреть все домашние задания для своей группы. Ученик может состоять в нескольких группах. Ученик может отправить решение домашнего задания.  
Решение домашнего задания включает в себя номер выполненного задания, дату выполнения, приложенный в виде ссылки или текста результат, состояние(ожидает проверки/на проверке/ожидает пересдачи/проверено), оценку и комментарий преподавателя в виде текста.  
Ученик может изменить отправленное решение домашнеего задания, если оно находится в состоянии "ожидает проверки" или "ожидает пересдачи".
Ученик может отправить столько решений домашнего задания, сколько указано попыток для этого домашнего задания.  

## Сущности в приложении:
- Преподаватель
- Ученик
- Группа учеников
- Урок
- Домашнее задание
- Решение домашнего задания
- Оценка

## Действия сущностей

### Преподаватель может:
- Создать домашнее задание
- Отменить домашнее задание
- Изменить текст и срок сдачи домашнего задания
- Изменять статус решения домашнего задания
- Добавить оценку решению домашнего задания
- Оставлять комментарии к решениям домашних заданий

### Ученик может:
- Отправить решение домашнего задания
- Изменить результат выполненного решения домашнего задания

## Ограничения

1. Домашнее задание включает в себя как минимум 1 попытку.
2. Ученик может отправить решение домашнего задания, только если домашнее задание предназначалось его группе.
3. Ученик может отправить решение домашнего задания, только если он отправил решений для домашнего задания меньше, чем допустимое количество попыток.
4. Ученик может отправить решение домашнего задания, только если домашнее задание не было отменено.
5. Ученик может изменить результат решения домашнего задания только если оно находится в состоянии "ожидает проверки" или "ожидает пересдачи".
6. Преподаватель может изменять только те домашние задания, которые он сам назначил.
7. Преподаватель может изменять статус, добавлять оценки и оставлять комментарии только для тех выполненных домашних заданий, для которых домашнее задание он сам назначил.
8. Преподаватель может изменить статус решения домашнего задания, только если оно уже не находится в статусе "проверено".
9. Преподаватель может добавить или изменить оценку только для того решения домашнего задания, которое находится в статусе "проверено".

## Информация о сущностях

### Преподаватель
- Id
- ФИО

### Группа учеников
- Id
- Наименование

### Ученик
- Id
- ФИО
- Коллекция Id групп учеников

### Урок
- Id
- Наименование

### Домашнее задание
- Id
- Дата назначения
- Дата сдачи
- Id преподавателя
- Id целевой группы учеников
- Заголовок
- Текст задания
- Приложенные материалы(опционально)
- Количество попыток для сдачи

### Оценка
- Id
- Значение

### Решение домашнего задания
- Id
- Id домашнего задания
- Id выполнившего ученика
- Дата последнего изменения
- Результат
- Состояние
- Комментарий препоадавтеля(опционально)
