# Микросервис для управления домашними заданиями для учеников

В данном микросервисе преподаватель может назначать группе учеников домашние задания. Домашнее задание включает в себя текст задания, приложенные ссылками материалы, срок сдачи, количество попыток и назначенную группу учеников.  
Преподаватель может изменить текст и срок сдачи, а так же полностью отменить домашнее задание.  
Преподаватель может просматривать выполненные домашние задания учеников.

Так же в данном микросервисе ученик, состоящий в определенной группе, может просмотреть все домашние задания для своей группы. Ученик может состоять в нескольких группах. Ученик может отправить выполненное домашнее задание.  
Выполненное домашнее задание включает в себя номер выполненного задания, дату выполнения, приложенный в виде ссылки или текста результат, состояние(ожидает проверки/на проверке/ожидает пересдачи/зачтено/не зачтено) и комментарий преподавателя в виде текста.  
Ученик может изменить отправленное выполненное домашнее задание, если оно находится в состоянии "ожидает проверки" или "ожидает пересдачи".
Ученик может отправить столько выполненных домашних заданий, сколько указано попыток для этого домашнего задания.  

## Сущности в приложении:
- Преподаватель
- Ученик
- Группа учеников
- Домашнее задание
- Выполненное домашнее задание

## Действия сущностей

### Преподаватель может:
- Создать домашнее задание
- Отменить домашнее задание
- Изменить текст и срок сдачи домашнего задания
- Изменять статус выполненного домашнего задания
- Оставлять комментарии к выполненным домашним заданиям

### Ученик может:
- Отправить выполненное домашнее задание
- Изменить результат выполненного домашнего задания задания

## Ограничения

1. Домашнее задание включает в себя как минимум 1 попытку.
2. Ученик может отправить выполненное домашнее задание, только если домашнее задание предназначалось его группе.
3. Ученик может отправить выполненное домашнее задание, только если он отправил выполнненных домашних заданий для него меньше, чем допустимое количество попыток.
4. Ученик может отправить выполненное домашнее задание, только если домашнее задание не было отменено.
5. Ученик может изменить результат домашнего задания только если оно находится в состоянии "ожидает проверки" или "ожидает пересдачи".
6. Преподаватель может изменять только те домашние задания, которые он сам назначил.
7. Преподаватель может изменять статус и оставлять комментарии только для тех выполненных домашних заданий, для которых домашнее задание он сам назначил.

## Информация о сущностях

### Преподаватель
- Id
- ФИО

### Группа учеников
- Id
- Наименование

### Ученик
- Id
- ФИО
- Коллекция Id групп учеников

### Домашнее задание
- Id
- Дата назначения
- Дата сдачи
- Id преподавателя
- Id целевой группы учеников
- Заголовок
- Текст задания
- Приложенные материалы(опционально)
- Количество попыток для сдачи

### Выполненное домашнее задание
- Id
- Id домашнего задания
- Id выполнившего ученика
- Дата последнего изменения
- Результат
- Состояние
- Комментарий препоадавтеля(опционально)
